# Overview

Wyre is a zero-dependency, high-performance web engine for Go, purpose-built for the era of AI agents, LLM token streaming, and real-time event applications. 

Unlike traditional Go web frameworks that wrap the standard library's `net/http` package, Wyre is built from the ground up to operate directly on raw TCP sockets. This architecture gives you extreme performance, granular control over networking lifecycles, and native streaming capabilities with minimal overhead.

## Key Features

- **Direct Raw Socket Architecture**: Bypasses the Go standard `net/http` server implementation. Connection timeouts, concurrency limits, and protocol parsing are handled directly on the raw TCP socket (`net.Conn`).
- **Memory Recycling Pool**: Implements automated request object recycling to achieve near-zero garbage collection (GC) pressure, even under high-throughput request rates.
- **AI-Native Concurrency Controls**: Provides route-level concurrency limiting and request queuing to handle bursty parallel tool calls generated by LLM agents.
- **Server-Sent Events (SSE) with Backpressure**: Built-in support for SSE token streaming. Backpressure is naturally handled via TCP socket window blocking, preventing server memory bloat when streaming to slow clients.
- **Unified Streaming Abstraction**: Exposes a transport-agnostic `Stream` interface that abstracts standard HTTP chunked responses, SSE, and raw upgraded/hijacked connections.
- **Model Context Protocol (MCP)**: Native support for MCP's Streamable HTTP transport, enabling developers to build and host high-performance MCP tools, prompts, and context servers.
- **Standard Library Compatibility**: Includes simple adapters to wrap standard Go `http.Handler` functions so you can integrate legacy code seamlessly.

## Quick Start Example

Below is a complete example showing how easy it is to define a router, register handlers, apply middleware, and launch a Wyre server:

```go
package main

import (
    "wyre"
)

func main() {
    // 1. Initialize a new router
    router := wyre.NewRouter()

    // 2. Register global middleware
    router.Use(wyre.Recovery())
    router.Use(wyre.CORS())

    // 3. Register route handlers
    router.HandleFunc("GET", "/hello", func(w *wyre.ResponseWriter, r *wyre.Request) {
        w.WriteFixedBody(200, "text/plain", []byte("Hello, Wyre!"))
    })

    // 4. Start the server on port 8080
    config := wyre.DefaultConfig("0.0.0.0:8080")
    server := wyre.NewWithConfig(config)
    
    server.ListenAndServe()
}
```
